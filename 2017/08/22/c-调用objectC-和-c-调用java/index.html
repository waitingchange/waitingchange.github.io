c++如何调用objectC

ios中 c++ 的文件可以用 .cpp 和 .mm 来结尾
区别是：
.mm ：带有这种扩展名的源代码文件，除了可以包含objective-c和C代码以外还可以包含C++代码。
.cpp：只能编译C++

所以，如果要从c++ 中调用 ios 底层的内容， .mm 文件必不可少。

由此可见 c++ 调用object-C 比较容易，更改后缀为.mm 就可以使用
